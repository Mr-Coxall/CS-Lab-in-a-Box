# Computer Science Lab in a Box

A single box that can be used to run High School Computer Science class, locally. Using TrueNAS Scale as the OS and then running docker containers on it to host:

- Gitea (remote git server)
- Coder IDE (web based VS Code server)
- Nginx (web server)
